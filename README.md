# College Management System

A **College Management System** built using **Django** to manage academic and administrative activities such as student records, faculty management, courses, attendance, and results.  
This project is designed for educational purposes and demonstrates how a real-world college ERP system can be implemented using Django.

---

## Features

- User Authentication (Admin, Faculty, Student)
- Student Management
- Faculty Management
- Course & Subject Management
- Attendance Management
- Result / Grade Management
- Dashboard for different user roles
- Secure login and role-based access

---

## Tech Stack

- **Backend:** Django (Python)
- **Frontend:** HTML, CSS, Bootstrap
- **Database:** SQLite (default, can be changed to MySQL/PostgreSQL)
- **Authentication:** Django built-in authentication system
